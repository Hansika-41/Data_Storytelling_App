#  Netflix Data Storytelling App

An application that presents meaningful insights through data storytelling using Streamlit and Plotly.

## Storyline Overview
- **Dataset Introduction**: Brief history of Netflix's transition to streaming data.
- **Exploratory Data Analysis (EDA)**: Comparing global volumes of movies versus episodic TV shows.
- **Visualizations**: Dynamic pie charts and timeline trend lines showing industrial shifts.
- **Insights & Findings**: Identification of rapid TV show production acceleration post-2015.
- **Final Conclusion/Recommendations**: Strategic data-driven advice for streaming content pipelines.
